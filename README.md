# Dev Task

1. Create math class/function which contains the following methods:
   - sum
   - sum multiple
   - subtract
   - multiply
   - divide
2. Write two solutions one in TS and one in JS

## Getting Started

1. Clone project
2. Install dependencies

```
npm install
```

3. Run vanilla js with:

```
npm run vanilla
```

3. Run typescript with:

```
npm run typed
```
